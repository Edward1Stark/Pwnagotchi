# Pwnagotchi Setup Bundle

Everything you need to get your Pwnagotchi up and running — OS image, drivers, and an automated internet connection script. Basically all the files I have mentioned in my blog about setting up you Pwnagotchi

## 📦 What's inside

- Raspberry Imager
- Required driver files
- `.ps1` — PowerShell script to share your PC's internet with the Pwnagotchi

## 📝 Detailed guide

For step‑by‑step instructions, troubleshooting, and tips, check out my blog:  
**https://edwardstark.website/setup-your-pwnagotchi/**
and you can download the OS files from here I was soo tired for dealing with large file on my uploader host
**https://github.com/jayofelony/pwnagotchi/releases/tag/v2.9.5.4**

## ⚠️ Requirements

- Windows 10/11
- PowerShell run as Administrator
- USB cable that supports data transfer
- New SD card preferably 32GB ( Believe me when I say new one is needed, you don't want to deal with the problems of writing the OS on a used SD card )
